Loja de Teste

Esta página simula o site de um cliente com o tracker instalado.

Clique no botão abaixo — o tracker vai anexar [Ref: TRK-...] na mensagem automaticamente:

Falar no WhatsApp
Como verificar:
  1. Abra o DevTools (F12) → Application → Local Storage → veja a chave _saas_trk_id.
  2. Passe o mouse no botão do WhatsApp → o link já contém [Ref: TRK-...].
  3. No painel, abra Leads do Demo Tenant → o seu acesso aparece como um lead novo.
